Файл состоит из подготовки базы знаний и подготовка пользовотельских запроов.
Ключевые технологии:

Python 3.x

Sentence Transformers (Hugging Face)

NumPy

Косинусное сходство для векторного поиска

Основные компоненты системы:

DataPreparation – модуль предобработки данных:

Разбивает документы на фрагменты с перекрытием (chunking)

Создает векторные представления текста с помощью моделей Sentence Transformers

Сохраняет базу знаний в формате JSON (тексты) и NPY (векторы)

SearchSystem – модуль семантического поиска:

Загружает предварительно созданную базу знаний

Векторизует пользовательские запросы

Находит наиболее релевантные фрагменты с помощью косинусного сходства

Возвращает отсортированные результаты с оценкой релевантности

AnswerGenerator – модуль генерации ответов:

Интегрируется с LLM API (заглушка в текущей реализации)

Формирует контекст на основе найденных фрагментов

Генерирует ответ на вопрос пользователя

RAGSystem – основной координационный класс:

Объединяет поисковую и генеративную компоненты

Предоставляет единый интерфейс для вопросно-ответного взаимодействия

Возвращает структурированные результаты с источниками
